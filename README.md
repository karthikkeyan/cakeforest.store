# cakeforest.store

Production website for [Cake Forest](https://cakeforest.store) — a bakery in KK Nagar, Madurai, Tamil Nadu.

## About

This repo contains the live website published at **cakeforest.store** (domain registered via Squarespace Domains).

Code is developed in the private [`cakeforest`](https://github.com/karthikkeyan/cakeforest) repo and promoted here when ready to publish.

## Stack

Plain HTML, CSS, and JavaScript — no build step required.

## Themes

- **Forest Canopy** — daytime theme (6 AM – 8 PM)
- **Neon Bazaar** — night-time theme (8 PM – 6 AM)

Theme switches automatically based on the visitor's local time.
