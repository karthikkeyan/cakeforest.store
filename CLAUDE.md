# CLAUDE.md — cakeforest.store

This is the **production** repo for the Cake Forest website, published at cakeforest.store.

## Workflow

- Development happens in the private `cakeforest` repo under `dev/website/`
- When ready to publish, production-ready files are promoted into this repo
- Pushing to `main` here deploys to cakeforest.store

## Structure

```
index.html        # Main entry point
css/main.css      # All styles (Forest Canopy + Neon Bazaar themes)
js/main.js        # Theme switching and interactions
data/menu.json    # Menu content
images/           # Website image assets
```

## Domain

- Registered at Squarespace Domains: cakeforest.store
- DNS should point to wherever this repo is hosted (GitHub Pages, Cloudflare Pages, etc.)
